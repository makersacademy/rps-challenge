# Rock, Paper, Scissors, Spock, Lizard
## Weekend Challenge
### Week Three
Completed 27 November 2016

Instructions
-------

* Run this app in Sinatra ```ruby app/rps.rb```
* Visit ```localhost:4567``` from a web browser
* Enter your name
* Make a selection
* See if you win or lose!
