# RPS Challenge

A basic web app of the classic Rock, Paper Scissor game.
Player enters name and plays best of 3 vs the computer


To run:
-------
```sh
$ git clone https://github.com/smasonmalik/rps-challenge.git
$ cd rps-challenge
$ bundle
$ rackup
```
