## Install
Try it out in your browser [here](https://arthur-r-p-s.herokuapp.com/).

Or to install:
``` bash
% git clone https://github.com/arthurfincham/rps-challenge
```

## Usage

Go to the project directory and install dependencies:

``` ruby
% bundle install
 ```

Run the app with Sinatra
``` ruby
% ruby app.rb
```
and you're off!
