How to play
----

To play this fabulous game, clone the repository as follows:

```sh
$ git clone git@github.com:[USERNAME]/rps-challenge.git
$ cd rps-challenge
$ bundle
$ rackup -p 4567
```

then navigate to http://localhost:4567/ to play!
