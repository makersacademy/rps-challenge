# Rock, Paper, Scissors challenge

Work done for the week 3 challenge for [Makers Academy](http://www.makersacademy.com)

## How to download and start the app
```sh
git clone https://github.com/[USERNAME]/rps-challenge.git
cd rps-challenge
bundle
rackup
```
