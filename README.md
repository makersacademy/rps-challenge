Instructions
-------
Installing:

```
$ git clone git@github.com:[USERNAME]/rps-challenge.git
$ cd rps-challenge
$ bundle
$ ruby app.rb -p 4567
```

In browser

1. Navigate to ```http://localhost:4567/```
2. Use on screen instrictions to play the game. Have fun!
