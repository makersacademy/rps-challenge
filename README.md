# Rock Paper Scissors

Welcome
-------

Welcome to this wonderful game of rock paper scissors.  It allows you, the player, to set your wits against the computer.  Bon chance mon ami!

How to use
-------

1. Fork this repo
2. Run ```bundle install```

3. Enter your name as prompted

4. Choose your move
