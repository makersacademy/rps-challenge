Instructions
-------

``` 
 $ git clone git@github.com:MarcoCode/rps-challenge.git  
 $ cd rps-challenge
 $ rackup
 
 From your browser visit localhost:4567 to play the game locally
 
```
* NOTE: I have used a global variable only to simulate a database
