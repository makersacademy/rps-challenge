# The age old game of RPS! #

**Setup**

Clone the RPS repo at git@github.com:Chrixs/rps-challenge.git, in terminal cd into the `rps-challenge` directory and run `ruby app.rb`. Open your browser of choice and type http://localhost:4567/ into the URL bar.

*The Game*

The game is simple, input your name on the home page, choose your weapon and the game plays itself out!
