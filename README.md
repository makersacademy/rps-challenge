# Rock Paper Scissors

![index](docs/img/index.png)

---------
### Functions

1. Enter player name
2. Choose rock, paper or scissors
3. View result
4. Go back (to play)

---------
### How to use

1. [Setup](docs/setup.md)
2. Run command `rackup`
3. In web browser, enter i.e. "localhost:9292" (or the port provided from previous step)

---------
